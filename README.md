# Sy-PackInst
Sy-PackInst is a terminal tool that is used to make it easier for users to install packages in their terminal.
[BASH ONLY!]
# How to install this tool?
The way to install this tool is to download 1timesy.sh (bash that is run once and will delete itself), copy and run this command
```bash
wget https://raw.githubusercontent.com/Syaifanisa/Sy-PackInst/main/1timesy.sh
```
or
``` bash 
curl -o 1timesy.sh https://raw.githubusercontent.com/Syaifanisa/Sy-PackInst/main/1timesy.sh
```
After that, you have to run the 1timesy.sh file, the task of this file is to install python 3 and curl automatically then create a folder called ~/Sy-PackInst and put the Sy-PackInst.py file in that directory, after that it automatically creates an alias Sy-PackInst in bashrc and then deletes itself, First of all we have to give permission to 1timesy.sh with chmod. (make sure you are in the writable directory)
``` bash
chmod +x 1timesy.sh
```
and then run 1timesy.sh
``` bash
./1timesy.sh
```
and done!, now you can use Sy-PackInst! 
# How to use the tool?
you just have to run this command to run the Sy-PackInst:
``` bash
Sy-PackInst
```
then choose the available options (reading more info is recommended), and enjoy! :)
